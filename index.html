Kay's Cheat Sheet

This cheat sheet is HTML made simple to help users navighate basic functions. HTML- hyper Text Mark Up Language is the backbone of webpages and it is integral to web development

HTML Basics

Elements and Structure

Name Tag Description
HTML Document <!DOCTYPE html> Defines the document type and version of HTML
Root Element <html> Encloses all HTML content
Head Section <head> Contains meta-information about the document
Title <title> Sets the title of the document displayed in the browser tab
Body Section <body> Contains the content of the document displayed in the browser
Anchor <a> Used to create hyperlinks in an HTML document
Target Attribute target="_blank" Used with the anchor element and tells the browswe to open the link in another tab
List Item <li> Used to create a list of items- they can be ordered or unordered
Ordered List <ul> These list are shown with a dot
Ordered List <ol> Each listed item is numbered by default
Paragraph <p> Contains text the user wants to display, like a block of text
Heading <h1><h6> These are used for heading, there are 6 types with h1 being the largest and h6 being the smallest
Line Break <br> This tells the browser to add a space below the last text before the br
Division <div> Used as a container that divides an HTML document into sections
Image <img> used to embed an image in documents. Used with src=" " that contains the image url (img has no closing tag)
Strong <strong> Used to make an element bold/ highlight seriousness
Table <table> Used to create a table in an HTML
Table Head <thead> Contains the heading of the table, this is where you enter the heading tag
Table heading <th> Used to enter the different labels of a table
Table Body <tbody> Contains information in the table
Table Row <tr> Used to create a row in the table
Table Data <td> Usually nested in tr to add the informaation in a cell
Table Footer <tfoot> Used to summarize the contents at the end of the table
